## вҡЎ Getting Started

### Prerequisites
```
Node.js v18+   вҖў   MongoDB Atlas account   вҖў   Cloudinary account   вҖў   Razorpay account
```

### 1пёҸвғЈ Clone the repository
```bash
git clone https://github.com/vishnu-13-97/Ecommerce

```

### 2пёҸвғЈ Backend Setup
```bash
cd backend
npm install
```

Create **`backend/.env`**:
```env
# в”Җв”Җ backend в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
PORT=5000
NODE_ENV=development

# в”Җв”Җ Database в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
MONGODB_URI=mongodb+srv://<user>:<pass>@cluster.mongodb.net/ecomm

# в”Җв”Җ Auth в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
JWT_SECRET=your_super_secret_key_here
JWT_EXPIRES_IN=7d

# в”Җв”Җ Cloudinary в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret

# в”Җв”Җ Email OTP (Gmail) в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
EMAIL_USER=your_gmail@gmail.com
EMAIL_PASS=your_16_char_app_password

# в”Җв”Җ Razorpay в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ
RAZORPAY_KEY_ID=rzp_test_xxxxxxxxxxxx
RAZORPAY_KEY_SECRET=your_razorpay_secret
```

```bash
npm run dev          # вҶ’ http://localhost:5000
```

## рҹ’і Razorpay Payment Flow

```
  User clicks "Place Order"
         в”Ӯ
         в–ј
  POST /api/user/orders  в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв–¶  Razorpay API
  { addressId, paymentMethod: "Razorpay" }         в”Ӯ
         в”Ӯ                                   Creates Order
         в”Ӯ  в—Җв”Җв”Җв”Җв”Җ { razorpayOrderId, amount } в”Җв”Җв”Җв”Җв”Җв”Җв”ҳ
         в”Ӯ
  Razorpay Checkout Modal Opens
  (UPI / Cards / Net Banking)
         в”Ӯ
  Payment Completed by User
         в”Ӯ
  POST /api/user/orders/verify
  { razorpay_payment_id,
    razorpay_order_id,
    razorpay_signature }
         в”Ӯ
  Backend verifies HMAC signature
         в”Ӯ
  вң… Order saved  вҶ’  Navigate to /order-success/:id
```
